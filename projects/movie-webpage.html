Preview of the Project
Code of the Project

Prompt: Create a movie webpage about almost anything using your knowledge of HTML and CSS with your partner.

Title: Escape.

Context: with my partner Joey, we used HTML and made a website about a pretend movie, and styled it with our knowledge about CSS.

Reflection: The most challenging part for me was thinking about a movie idea. I had so many ideas me and Joey wanted to do, and could only choose one. It was also challenging to figure out the color scheme for the website. I needed to find a color scheme that was well-fitted with the movie idea I chose. I think the biggest takeaway I got from this project was creativity skills. This project encouraged me to think outside of the box and created a website that was well-organized and pleasing to the eye. Not only that, I also gained communication skills. This project was a pair project, and me and my partner Joey had to communicate in order to create the final result. In the near future, I'll be looking foward to being able to create more websites.
