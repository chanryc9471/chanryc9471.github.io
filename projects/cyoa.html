Life as a Baby CYOA


Image of CYOA Start *


Context

The project is about making a choose-your-own-adventure project that will practice skills on linking markdown files through working collaboratively on GitHub. The project will walk through how a day in the life as a baby feels like, with decisions that you make, which will alter the outcome.

Content

Me and my partner Wendy made a choose-your-own-adventure project using our knowledge of linking markdown files on the topic of "Life as a Baby."

Reflection

I enjoyed making this project with my partner, although there were some challenges. The most challenging part for me was linking the markdown files. Sometimes the filepaths weren't linked properly and went to a 404 error page. Thankfully, the issues were fixed so it worked properly. There were many takeaways I got from this project, but the major one was collaboration. Me and my partner communicated consistently, and made sure to inform one another if they needed to git pull the changes. We also took into account of one another's ideas, and combined our ideas for the projects to create one whole idea. My next steps will be to use what I learned from this project about linking markdown files in future projects.
